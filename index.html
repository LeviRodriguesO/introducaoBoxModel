Box model

Box Model é importante nos layouts de sites web, ele facilita a aplicação do CSS.

O que o Box Model

  • É Cada elemento reprensentado como uma caixa retangular.
  • É uma caixa que possui duas dimensões (Largura e Altura).

Propriedades da caixa

  • Tamanho (Largura x altura) width x height
  • Conteúdo / content
  • Preenchimento interno / padding
  • Bordas / Bordes
  • Espaços fora da caixa / Margin

Margin

Margin, é o espaço (margem entre os elementos)

  • Podemos dividir ela em quatro valores
  • Values: <lenght>, <percentage> e auto. Geralmente usamos uma forma abreviada shorthand para descrever o margin. Esse formato segue o sentido horário começando pelo top, depois right, bottom e left.
  • O margin é aplicado em elementos com display block.
  • Cuidado com o margin collapsing que é quando o top se junta ao bottom.

Padding

O padding é o preenchimento interno da caixa.

  • A propriedade padding pode ser escrita como nos formatos apresentados abaixo:
  • Geralmente usamos a forma abreviada (shorthand para escrever o padding). Ele segue no sentido horário, começando pelo top, depois right, bottom e left.
  • O padding pode ter valores de comprimento (rem, em, px) ou porcentagem (%).
  • O padding poderá causar diferença na largura de um elemento.

Box Sizing

O navegador não define o tamanho dos elementos de forma muito conveniente, mas nós podemos sobscrever esse comportamento com a propriedade box-sizing.

Conteúdo

Essa propriedade faz faz o navegador respeitar as medidas especificadas do widht e height do nosso elemento

Dica!

Como é muito útil trabalhar como o box-sizing usando o valor border-box é uma boa prática aplicá-lo na página toda.

Agora todos os elementos da página irão respeitar os tamanhos especificados.

Display-block-inline

Display: block; VS Display: inline;

  • Como as caixas se comportam em relação as outras caixas
  • Comportamento externo das caixas

Display block

  • Ocupa toda, linha colocando o próximo elemento abaixo desse
  • widht e height são respeitadas
  • padding, margin, border irão funcionar normalmente

Display inline

  • Os elementos ficam ao lado do outro e não empurram outros elementos para baixo
  • widht e height não funcionam
  • somente valores horizontais de margin
